# 🛒 Superstore Sales Analysis (Business Intelligence Project)

## 📌 Overview
This project analyzes retail sales data to uncover insights about revenue, profitability, customer behavior, and regional performance.

---

## 🎯 Objectives
- Identify top-performing categories and products  
- Detect loss-making segments  
- Analyze regional and customer trends  
- Evaluate discount impact on profit  

---

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy)  
- Matplotlib & Seaborn  
- Jupyter Notebook  

---

## 📊 KPIs
- Total Sales  
- Total Profit  
- Profit Margin  
- Number of Orders  

---

## 🔍 Key Insights
- Technology is the highest-performing category  
- High discounts reduce profitability significantly  
- A small number of customers generate most revenue  
- Some sub-categories consistently generate losses  

---

## 🚀 Business Recommendations
- Focus on high-profit categories  
- Reduce excessive discounting  
- Improve low-performing products  
- Retain high-value customers  

---

- Notebook with full analysis  
- Dataset  
- Visualizations
